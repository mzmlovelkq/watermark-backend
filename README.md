## 项目结构

```
ToolApi
├── config -- 工具类及通用代码
├── conrtoller -- 系统接口
├── core -- 核心异常拦截
├── exception -- 断言类及自定义异常
├── mapper -- 数据映射
├── model -- 实体
└── service -- 服务逻辑处理
```
